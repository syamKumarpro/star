# NumTrack Pro 🔢

A powerful, offline-first number tracking and pattern analysis web application.

## 🚀 Live Demo

> Deployed via GitHub Pages: `https://<your-username>.github.io/<repo-name>/`

## 🔐 Login

- **Username:** Any username works

## ✨ Features

- 📊 Track numbers by date with a clean numeric keypad
- 📈 Pattern analysis — frequency, streaks, co-occurrence
- 🔮 Predictions based on historical data
- 🎯 Range matching tool
- 📤 Export / Import your data as JSON
- 🌙 Fully dark-themed, offline-first (data stored in browser localStorage)

## 📁 File Structure

```
/
├── index.html        ← Login screen (entry point)
├── app.html          ← Main NumTrack Pro application
└── README.md         ← This file
```

## 🛠 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Click **Save** — your site will be live in ~1 minute

## 🔒 Privacy

All data is stored locally in your browser (`localStorage`). Nothing is sent to any server.
